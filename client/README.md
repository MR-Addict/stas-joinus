# 校大学生科协报名表单前端

## 1. 开发项目

> 注意：
>
> 开发前配置好环境变量，变量内容可参考根目录的 [.env.development.local.example](.env.development.local.example) 文件。

安装依赖：

```sh
npm install
```

开发项目：

```sh
npm run dev
```

编译项目：

```sh
npm run build
```

预览项目：

```sh
npm run preview
```

格式化代码：

```sh
npm run format
```

## 2. 部署项目

如何部署项目请阅读根目录的 [README](../README.md)
